Descripción del Sistema

Esta aplicación ejecuta pruebas de rendimiento comparativas entre bases de datos relacionales (PostgreSQL) y no relacionales (MongoDB). Se evalúan dos escenarios principales: escritura masiva (inserción concurrente de datos) y operaciones de consulta (búsquedas con diferentes niveles de complejidad). Los resultados se presentan mediante visualizaciones gráficas para facilitar el análisis de desempeño.

Arquitectura del Sistema
  • Backend (Flask): Servidor Python que ejecuta las pruebas de rendimiento y proporciona API REST con respuestas en formato JSON
  • Threading: Implementa 10 hilos concurrentes para simular escenarios de carga real en producción
  • PostgreSQL: Operaciones con garantías ACID, validación de integridad referencial mediante claves foráneas
  • MongoDB: Almacenamiento de documentos JSON desnormalizados para optimizar lecturas
  • Medición de Rendimiento: Utiliza time.time() para cronometraje con precisión de milisegundos
  • Visualización: Chart.js para renderizado de gráficas interactivas en tiempo real
Tipos de Operaciones de Consulta
  • Consulta Simple: Filtrado básico (WHERE temperatura > 25) limitado a 100 registros
  • Consulta con Rangos: Condiciones múltiples con operadores lógicos AND y BETWEEN
  • Operación de Agregación: Funciones de agregación (AVG, MAX, MIN) sobre el conjunto completo de datos
